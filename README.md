Sentiment Analysis Using Machine Learning (Twitter Dataset)
Project Overview:
This project focuses on building a Sentiment Analysis model capable of classifying tweets into three
categories:
Positive (1), Neutral (0), Negative (-1).
Objectives:
- Clean and preprocess tweet text
- Convert text into numerical vectors using TF-IDF
- Train a machine learning model
- Evaluate model performance
- Predict custom text sentiment
Dataset Information:
Columns:
- clean_text: tweet text
- category: -1 negative, 0 neutral, 1 positive
Workflow:
1. Load dataset
2. Clean data
3. Preprocess text
4. Train-test split
5. TF-IDF vectorization
6. Logistic Regression training
7. Evaluation
8. Custom predictions
Future Improvements:
- Deep learning models
- Web UI
- API deployment
